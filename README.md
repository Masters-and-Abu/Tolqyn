# Tolqyn

Tolqyn is an online streaming and podcasting service!

# How to contribute

- `git clone https://github.com/Masters-and-Abu/Tolqyn.git && cd Tolqyn`
- `npm i`
- Go to your folder
- Commit using `npm run commit`
